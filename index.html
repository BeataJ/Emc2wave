The most recent trends shows there is also big interest in NoSQL databases.

What is this?

Those type of databases don’t use sql language and don’t store data in traditional table format (rows and columns) but use different type of data forms like documents as the records. The classic example is Mongo database server that use javascript engine to query and store data (json format).

Servers

Operating Systems

As DevOps specialists we deal with different operating systems for servers and desktops.

We like variety of OS’s as they provide different tools and approach to system administration.

We use Windows, Mac OS, Linux on workstations to develop applications and help clients to administer their servers.

The recent versions of operating systems get more and more complex as new tools methodology are being introduced.

For example Linux recently introduced new binaries to manage processes called systemd and new type of system firewall named firewalld that change totally philosophy to processes management and firewall rules.

Also, Windows 2012 doesn’t seems to be easy to manage as new tools and security measures were introduced to protect system resources.

Also, we realize that system administrations doesn’t mean only periodic servers checks as some service providers do. This is a complex tasks to:

  • manage processes and applications on the servers,
  • manage users and their permissions to resources,
  • monitor users activity, network traffic and security,
  • manage security policies, install patches and updates,
  • create backup of resources,
  • project and advise for system grows.
